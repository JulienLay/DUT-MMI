Les media queries

Exemple de media query :

@media only screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Comprendre les media queries

Mettre en place la media query ci-dessus dans votre style.css.

Avec l'aide de cette page de documentation du W3C sur les media queries, expliquer avec un commentaire dans style.css.
